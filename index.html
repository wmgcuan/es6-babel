Hello ECMA Script 6
babel src/index.js -o dist/index.js

注：
1 dist文件夹需手动创建
2 要运行babel命令需全局安装：cnpm install -g babel-cli
3 本地安装运行命令：
"scripts": {
"build": "babel src -d dist"
}
4 只能做基础转换
当使用import './mod.js'时，会转换成require('./mod.js')
模块依赖时请使用webpack

babel命令
# 转码结果输出到标准输出
$ babel example.js

# 转码结果写入一个文件
# --out-file 或 -o 参数指定输出文件
$ babel example.js --out-file compiled.js
# 或者
$ babel example.js -o compiled.js

# 整个目录转码
# --out-dir 或 -d 参数指定输出目录
$ babel src --out-dir lib
# 或者
$ babel src -d lib

# -s 参数生成source map文件
$ babel src -d lib -s
